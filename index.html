Why CMDAI2?

Everything you need for efficient programming in one place.

Auto Mode

A fully autonomous agent. Creates plans, edits code, runs tests, and solves problems independently.

Model Support

From powerful APIs (OpenAI, Groq, Mistral) to fully local GGUF models via llama-cpp-python.

Tool-calling

Secure execution of system commands, file editing, and real-time web searching.

IDE Integration

Automatic context retrieval from your code editor, allowing the AI to better understand your project.

Quick Start

Install CMDAI2 in minutes and start coding.

01

Requirements

Ensure you have Python 3.12 or newer installed on your system.

python --version
02

Install Dependencies

Clone the repository and install the required libraries.

pip install -r requirements.txt
03

Run the App

Launch the application and select your AI provider.

python main.py

Who is it for?

For those tired of AI subscriptions who seek a local, free (or cheap) coding assistant in the terminal. Perfect for offline work with GGUF models or using lightning-fast providers like Groq.

Notes & Limitations

  • Active development phase — bugs may occur.
  • Local mode requires separate GGUF model downloads.
  • Built with Python and the rich library for CLI interface.
